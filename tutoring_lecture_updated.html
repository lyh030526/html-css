폼(Form) & 입력 태그

사용자 입력을 받는 다양한 폼 요소들 — 코드와 직접 동작 확인

📬 <form> 태그 — 폼의 기본 구조

<form> 사용자 입력을 서버로 전송하는 컨테이너 — action과 method 속성이 핵심
<form action="/submit" method="post" > <input type="text" name="username"> <input type="submit" value="제출"> </form>
form 주요 속성
속성설명예시
action데이터를 보낼 서버 URLaction="/login"
method전송 방식 (get / post)method="post"
get vs post: get은 URL에 데이터가 노출됨 (검색에 사용). post는 숨겨서 전송 — 비밀번호·개인정보는 반드시 post!

🏷️ id / name / class 속성

속성역할특징
id 요소를 고유하게 식별 페이지 내 단 하나만 존재해야 함. CSS(#id), JS에서 사용
name 폼 데이터 전송 시 키(key) 역할 서버로 보낼 때 name=value 쌍으로 전달. 중복 가능
class 요소를 그룹으로 묶음 여러 요소에 같은 class 사용 가능. CSS(.class)에서 스타일 일괄 적용
id / name / class 비교 세 속성의 쓰임새 차이
<!-- id: 페이지에서 딱 하나, JS/CSS에서 식별 --> <input id="username" type="text" > <!-- name: 서버로 전송될 때 key값 --> <input name="username" type="text" > <!-- class: 여러 요소에 같은 스타일 적용 --> <p class="highlight">강조 문단 1</p> <p class="highlight">강조 문단 2</p>
활용 예시
#username → CSS/JS에서 id로 선택
name="username" → 서버에 username=홍길동 으로 전달
.highlight → CSS에서 class로 한꺼번에 스타일 적용
정리: id는 하나만 / name은 서버 전송용 키 / class는 CSS 스타일 묶음. 실무에서는 세 가지를 함께 쓰는 경우가 많아요.

display 속성

요소가 화면에 배치되는 방식을 결정하는 핵심 CSS 속성

📦 block vs inline vs inline-block

속성값특징너비/높이줄 바꿈대표 태그
block 가로 전체 차지 width/height 적용 O 자동 줄바꿈 div, p, h1~h6
inline 내용 크기만큼만 width/height 무시! 줄바꿈 없음 span, a, strong, em
inline-block 인라인처럼 나란히 width/height 적용 O 줄바꿈 없음 직접 설정 필요

💻 코드 예제

/* block → inline-block 변환 */ div { display: inline-block; width: 100px; height: 100px; border: 2px solid black; } /* inline: width/height 무시됨 주의! */ .inline { display: inline; width: 100px; /* ← 이건 무시됨 */ height: 100px; /* ← 이것도 무시됨 */ }

float & clear

요소를 왼쪽 또는 오른쪽으로 띄워 배치하는 속성

🌊 float 동작 원리

값설명
float: left요소를 왼쪽으로 띄움 → 다른 요소가 오른쪽을 채움
float: right요소를 오른쪽으로 띄움 → 다른 요소가 왼쪽을 채움
clear: left왼쪽 float 해제 (왼쪽 float 아래로 내려옴)
clear: right오른쪽 float 해제
clear: both양쪽 모두 float 해제 (가장 많이 사용)

👁️ 시각적 데모

float: left / right
right
left

float 해제 후 텍스트가 아래에 오게 됩니다 (clear: both)

텍스트가 float 옆을 채우는 예제
img

이렇게 이미지 옆에 텍스트를 배치할 때 float을 많이 사용합니다. 텍스트가 자동으로 이미지 옆을 감쌉니다.

💻 코드 예제

#box1 { float: right; margin-right: 10px; } #box2 { float: left; margin-right: 10px; } p { clear: both; /* float 해제! */ }
주의: float된 요소는 부모의 높이 계산에서 제외됩니다. 부모가 찌그러지는 문제가 생기면 부모에 overflow: hidden 또는 자식에 clear: both를 사용하세요.

position 속성

요소의 위치를 직접 지정하는 CSS 속성 — top, right, bottom, left와 함께 사용

📍 position 종류 비교

값기준점문서 흐름특징
static 없음 (기본값) 유지 top/left 등 무시됨
relative 자기 원래 위치 유지 원래 자리는 남아있음
absolute 가장 가까운 position 조상 이탈 조상에 position:relative 필요!
fixed 뷰포트(화면) 이탈 스크롤해도 고정됨
sticky 스크롤 위치 기반 유지 특정 위치 도달 시 fixed처럼 고정

🔍 relative vs absolute 핵심 차이

position: relative

relative
top:20 left:30
원래 자리 공간 유지

자기 원래 자리에서 상대적으로 이동. 원래 공간은 그대로 남아있음.

position: absolute

부모 (position: relative)
absolute
bottom right

부모(relative)를 기준으로 정확한 위치에 배치. 문서 흐름에서 이탈.

📌 fixed vs sticky

position: fixed

뷰포트(브라우저 창)를 기준으로 고정. 스크롤을 해도 항상 같은 위치에 있음.

예: 네비게이션 바, 채팅 버튼, 상단 고정 헤더

position: sticky

평소에는 relative처럼 동작하다가, 지정한 위치(top: 10px)에 도달하면 fixed처럼 고정.

예: 스크롤 시 상단에 달라붙는 목차, 컬럼 헤더

💻 코드 예제

/* relative + absolute 조합 (가장 많이 쓰는 패턴!) */ .parent { position: relative; /* 기준점 설정 */ width: 300px; height: 300px; } .child { position: absolute; /* 부모 기준으로 배치 */ right: 30px; bottom: 30px; } /* fixed: 화면 우하단 고정 버튼 */ .fixed-btn { position: fixed; bottom: 20px; right: 20px; } /* sticky: 스크롤 시 상단 고정 */ .sticky-header { position: sticky; top: 0; }
absolute 사용 패턴: absolute를 사용할 때는 반드시 부모 요소에 position: relative를 설정하세요. 그렇지 않으면 body를 기준으로 배치되어 예상치 못한 위치에 놓입니다.

z-index & visibility

요소의 쌓임 순서와 보임/숨김을 제어하는 속성

🗂️ z-index — 쌓임 순서

특징설명
숫자가 클수록위(앞)에 표시됨
음수 가능배경 뒤로 보낼 수 있음
position 필요static이 아닌 position에서만 동작
#box1 { z-index: -20; } /* 가장 뒤 */ #box2 { z-index: 20; } /* 가장 앞 */ #box3 { z-index: 10; } /* 중간 */
시각적 데모
box1
z-index: -20
box2
z-index: 20
box3
z-index: 10

👁️ visibility vs display:none 차이

visibility: hidden

요소가 보이지 않지만, 자리(공간)는 그대로 유지됩니다.

보임
← 자리 있음
보임

display: none

요소가 완전히 제거되어 자리(공간)도 사라집니다. 다른 요소가 그 자리를 채웁니다.

보임
바로 붙음
#box1 { visibility: visible; } /* 보임 (기본값) */ #box2 { visibility: hidden; } /* 숨김 (자리는 유지) */ /* display:none 은 자리까지 사라짐 */ #box2 { display: none; }
정리: visibility: hidden = 투명 망토 (자리 차지 O) / display: none = 완전 삭제 (자리도 없음)
